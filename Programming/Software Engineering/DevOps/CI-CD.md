```yaml
  - create Docker image
```

```yaml

  - unit tests

  - contract tests
  - security scanning (SAST)
```

```yaml
deploy:
  staging:
    - deploy to staging
    - run E2E tests
    - run performance tests
    - manual approval gate

    - smoke tests
    - monitor metrics
    - auto-rollback on failure
```
