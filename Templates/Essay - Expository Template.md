## Self-Review Checklist
- [ ] Remains objective (no "I think" or personal opinions)
- [ ] Information is accurate and well-researched
- [ ] Explanations are clear and logical
- [ ] Technical terms are defined when necessary
- [ ] Examples make abstract concepts concrete
- [ ] Flows logically from simple to complex
- [ ] Conclusion summarizes without adding new info
- [ ] Formal, informative tone

## Useful Expository Phrases
| Function | Phrases |
|----------|---------|
| Defining | "X can be defined as...", "In essence, X refers to..." |
| Explaining | "This occurs because...", "The mechanism involves..." |
| Exemplifying | "For instance,...", "This is illustrated by..." |
| Sequencing | "Initially,...", "Subsequently,...", "Finally,..." |
| Clarifying | "In other words,...", "To put it simply,..." |
| Concluding | "In summary,...", "To conclude,..." |
