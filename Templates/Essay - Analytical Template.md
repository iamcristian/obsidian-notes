## Self-Review Checklist
- [ ] Thesis is analytical (HOW/WHY, not just WHAT)
- [ ] Each paragraph focuses on ONE element
- [ ] Evidence is specific and well-chosen
- [ ] Analysis goes beyond summary/description
- [ ] Connections between elements are explored
- [ ] Synthesis brings elements together meaningfully
- [ ] Conclusion offers deeper insight
- [ ] Academic, objective tone

## Analytical Verbs to Use
| Instead of... | Use... |
|---------------|--------|
| shows | demonstrates, illustrates, reveals |
| says | argues, contends, asserts, suggests |
| is about | explores, examines, interrogates |
| uses | employs, utilizes, deploys, leverages |
| means | signifies, implies, connotes, represents |

## Useful Analytical Phrases
| Function | Phrases |
|----------|---------|
| Introducing evidence | "This is exemplified by...", "As evidenced in..." |
| Analyzing | "This suggests that...", "This reveals..." |
| Interpreting | "This can be understood as...", "This implies..." |
| Connecting | "This connects to...", "Similarly,...", "In contrast,..." |
| Synthesizing | "Taken together,...", "These elements converge to..." |
