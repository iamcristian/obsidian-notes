## Causal Language
| Indicating Cause | Indicating Effect |
|------------------|-------------------|
| Because of... | As a result,... |
| Due to... | Consequently,... |
| Owing to... | Therefore,... |
| On account of... | Thus,... |
| This stems from... | This leads to... |
| X is caused by... | X results in... |
| The reason for... is... | The effect of... is... |
| X contributes to... | This gives rise to... |

## Causal Verbs
| Cause verbs | Effect verbs |
|-------------|--------------|
| causes | results from |
| produces | stems from |
| generates | arises from |
| triggers | is a consequence of |
| brings about | follows from |
| leads to | is attributed to |
